# masters-application-tracker
a project to track my applications made to universities


a full-stack web app with Firebase authentication and Firestore database, deployed on GitHub Pages. Users can securely sign up/login, then manage graduate school applications through a dashboard displaying real-time statistics, status distribution charts, and deadline timelines. The application features complete CRUD operations with search/filter functionality, document tracking, and responsive design across all devices. The tech stack includes HTML5/CSS3, JavaScript, Chart.js for visualizations, Firebase 9.6.1 for backend services, and GitHub Pages for hosting, with Firestore security rules ensuring data isolation. Setup takes 5 minutes: clone the repo, configure Firebase credentials, enable Authentication/Firestore, and deploy via GitHub Pages
